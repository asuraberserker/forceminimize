# 窗口守护器（Windows 11）

这个项目提供一个 Python 桌面程序：

- 启动后弹出 GUI 窗口，列出当前系统已打开窗口标题。
- 选择一个目标窗口并点击“开始守护”。
- 如果当前激活窗口不是目标窗口：
  - 自动将目标窗口最小化。
  - 将目标窗口对应进程静音（通过 Windows 音频会话）。
- 当切回目标窗口时：
  - 自动恢复目标窗口（如果处于最小化）。
  - 自动取消静音。

## 运行环境

- Windows 11
- Python 3.9+

## 安装依赖

在 PowerShell 或 CMD 中执行：

```bash
pip install -r requirements.txt
```

## 启动程序

```bash
python window_guard.pyw
```

## 使用说明

1. 点击 **刷新窗口列表**。
2. 在下拉框中选择你希望守护的窗口。
3. 点击 **开始守护**。
4. 程序会每 0.5 秒检测一次当前前台窗口：
   - 前台不是目标窗口 → 最小化 + 静音目标窗口。
   - 前台是目标窗口 → 恢复（若最小化）+ 取消静音。
5. 点击 **停止守护** 可暂停行为。

## 说明与限制

- 静音控制基于“进程音频会话”：
  - 如果一个窗口所属进程存在音频会话，会被设置静音/取消静音。
  - 如果该进程当前没有音频会话，静音调用不会产生可见效果。
- 某些系统窗口（或权限较高进程窗口）可能无法被控制。
- 请尽量以普通桌面应用窗口作为目标。

## 常见问题

### 1) 找不到目标窗口？
先确认窗口已经打开并可见，然后点击“刷新窗口列表”。

### 2) 静音不生效？
确认该应用确实正在播放声音，并且未被系统/驱动层面重定向到其他音频设备。

### 3) 程序只能在 Windows 跑吗？
是的，本程序使用了 Windows API（`pywin32`）和 Windows 音频会话接口（`pycaw`）。
